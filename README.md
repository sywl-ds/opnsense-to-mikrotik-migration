# OPNsense to MikroTik Migration

## Purpose

This repository documents the migration process from OPNsense firewall
to MikroTik router (hEX) with minimal downtime.

## Scope

- Firewall
- NAT
- VPN
- VLAN
- Internet Gateway

## Device

- Old Router: OPNsense
- New Router: MikroTik hEX
- RouterOS: v7.x

## Migration Strategy

- Parallel deployment
- Gradual cut-over
- Rollback-ready
